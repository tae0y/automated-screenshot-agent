# README

## Getting Started

1. Get the code

    ```bash
    git clone {REPO_URL}
    ```

1. Activate uv environment.

    ```bash
    uv venv .venv

    # Windows
    .venv\Scripts\activate

    # Unix
    source .venv/bin/activate
    ```

1. Run the code

    ```bash
    uv run src/main.py
    ```
